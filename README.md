# FullSensorIrrigation (FSI) System on Arduino

## About
This project was one that I had to make for the Computer Architecture / Embedded Systems course. 
It is built entirely on a single Arduino and one .ino program. The complete list of items is as follows:

- 1x Soldering Iron

- 1x Solder spool

- 1x N95 Facemask
